
# # NORDSEE-PARSER  
  
**Nordsee-Parser** created to parse vacancies data from https://karriere.nordsee.com/de/  
  
  
## Used packages  
- PyQuery  
- Requests  
- Splinter
  
## Info  
**parser.py** include main parsing functions
**exchanger.py** include main pasting functions
  
## Installation & start  
  
    $ make venv   
    $ source venv/bin/activate  
    $ make requirements  
    $ python parse.py 
    $ python exchanger.py
