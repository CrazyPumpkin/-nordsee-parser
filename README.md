# # NORDSEE-PARSER

**Nordsee-Parser** created to parse vacancies data from https://karriere.nordsee.com/de/


## Used packages
- PyQuery
- Requests

## Info
**parser.py** include main functions

## Installation & start

    $ make venv 
    $ source venv/bin/activate
    $ make requirements
    $ python parse.py
